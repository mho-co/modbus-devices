## Método de leitura
Não é necessário deslocar os endereços. Sem necessidade de alterar formato de leitura.
- Deslocamento de registradores: `0`;
- Byte-swap: `não`;
- Word-swap: `não`;
- DoubleWord-swap: `não`.
